# ChatGpt SpringBoot integration

A REST service that uses ChatGPT with Assistant and Files for text chat.

### Functionalities:
- Assistant
- File
- Thread Bot

#### Important!
Don't forget to add your **chatKey** and **assistantKey** values in the **application.properties** file.
